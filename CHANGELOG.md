# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-18

### Added
- **Sprint 8 (Production Readiness)**: Implemented advanced SEO (JSON-LD, full Open Graph), dynamic sitemap, robots.txt, accessibility (Skip to Content, focus states), error handling (404, loading, error boundaries), and comprehensive documentation.
- **Sprint 7 (Recruiter Experience)**: Created the `/contact` route serving as a premium recruiter dashboard featuring availability, resume downloads, FAQ, and trust indicators.
- **Sprint 6 (Proof of Engineering)**: Created the `/engineering` route acting as an engineering handbook detailing workflows, architectural thinking, and quality standards.
- **Sprint 5 (Technical Toolbox)**: Created the `/skills` route to showcase technologies categorized dynamically, alongside certifications and active learning goals.
- **Sprint 4 (About & Journey)**: Built the `/about` route detailing Samarth's engineering story, values, and educational background.
- **Sprint 2.5 (Content Integration)**: Decoupled all data from UI into a strict `content/` architecture.
- **Sprint 0-2 (Foundation)**: Initialized Next.js App Router, Tailwind CSS, dark mode support, and the landing page experience.

### Changed
- Transitioned from static hardcoded components to a fully modular, Content-Driven UI architecture.
